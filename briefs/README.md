# briefs/

Короткие записки и черновики для работы с Claude в чате. Менее формальные чем `tasks/`.

## Что сюда

- Мысли Dr. Nik «надо подумать про X»
- Черновики идей, которые ещё не оформлены в ТЗ
- Заметки с обсуждения, на которые потом ссылается Claude в сессии
- Промпты-эксперименты для отладки
- Скопированные куски документации внешних сервисов

## Что НЕ сюда

- Формальные ТЗ для Code-агента — в `tasks/active/`
- Research-документы — в `docs/research/`
- Эфемерное (живёт одну сессию) — в `tmp/` (gitignored)

## Срок жизни

Brief может либо превратиться в формальную задачу (перенести в `tasks/active/<имя>.md`), либо устареть (перенести в подпапку `briefs/archive/` или удалить).
