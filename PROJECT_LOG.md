## Текущая версия
**v0.2.0** — добавлены публикация в VK (сейчас + по расписанию через APScheduler), вкладка «Медиа» (drag-n-drop, генератор промптов для внешних AI), 4 новые рубрики (включая свободную тему), поддержка видео в постах, soft-delete, дублирование, иконочные кнопки, модалка подтверждения, очередь тостов, hot-keys. Готова к тестированию.

## Что дальше (план)
- **v0.3** — контент-план на месяц (календарь), аналитика VK, генерация видео через Kling Video API
- **v0.4** — деплой на VPS, Docker, GitHub Actions, миграция на PostgreSQL

## Как добавить новую сессию
1. Скопировать [`docs/sessions/_template.md`](docs/sessions/_template.md) в файл `docs/sessions/YYYY-MM-DD-session-N.md`.
2. Заполнить разделы (цель, что сделано, решения, проблемы, идеи).
3. Добавить строку в таблицу выше и в `docs/sessions/README.md`.
4. Закоммитить и запушить.
