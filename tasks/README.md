# tasks/

Формализованные ТЗ для Claude Code-агента (Discovery / Implementation / Fix).

## Структура

- `tasks/active/` — ТЗ под текущую или ближайшую сессию. Один файл за раз, чтобы Code-агент не запутался в контексте.
- `tasks/archive/` — все закрытые ТЗ. Хранятся для истории и анализа уроков.

## Формат ТЗ

Шаблон описан в `CLAUDE.md` (раздел 5). Кратко: тип задачи, контекст, цель, не-цели, decision gates, шаги, сценарии падения, финальный артефакт.

Длина: ≤300 строк норма, >500 — обязательная декомпозиция.

## Отличие от `briefs/`

- **`briefs/`** — короткие записки для Claude в чате (1-2 страницы, неформальные)
- **`tasks/`** — формальные ТЗ для Claude Code-агента (decision gates, шаги с проверками)
