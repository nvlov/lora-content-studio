# docs/research/

Research-документы по внешним интеграциям и API.

Каждое исследование — отдельный `<integration>-research.md` со структурой из `CLAUDE.md` (раздел 4.1):
- Источники (URL + дата + тип)
- Что подтверждено железно
- Что подтверждено косвенно
- Unknowns
- Главный вопрос
- Вердикт (GREEN / YELLOW / RED)
- План закрытия unknowns (если YELLOW)

Discovery обязателен перед любой реализацией внешней интеграции.
