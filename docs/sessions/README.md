# Журнал сессий разработки

В этой папке — детальный отчёт по каждой рабочей сессии: что сделано, какие решения принимались, что не удалось, идеи на следующий раз.

Файлы именуются: `YYYY-MM-DD-session-N.md`

|---|---|---|---|

## Шаблон новой сессии

Скопируйте `_template.md` в новый файл с именем `YYYY-MM-DD-session-N.md` и заполните разделы.
