# references/

Бренд-материалы и референсы, на которые опирается генерация контента — но которые не являются кодом.

## Что сюда класть

- Тон голоса Лоры: словарь характерных слов и выражений, что Лора говорит / не говорит
- Гайдлайны рубрик: точное описание формата каждой из 8 рубрик (см. `core/generators/prompts.py`), примеры лучших постов
- Бренд-голос LinguaAI: как мы говорим про продукт, что обещаем, что нет
- Style guide на английский (для Kling-промптов): какие визуальные приёмы используем, что избегаем
- Примеры эталонных постов (best-of), на которые можно ссылаться при отладке промптов

## Что не сюда

- Картинки Лоры (модельные эмоции) — в `assets/lora/`
- Сгенерированные пользователем медиа — в `static/uploads/`
- Внешние видео-материалы из Kling — в `data/inbox/video/`
- Рабочая БД — в `data/lora.db`

## Зачем

Когда промпт-генератор будет научен «читать» эти референсы (через RAG или просто injection в системный промпт), все тексты станут когерентнее с брендом. Сейчас файлы здесь — это форма документации, которую Dr. Nik ведёт руками.
